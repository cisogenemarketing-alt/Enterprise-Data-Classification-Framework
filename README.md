# Enterprise-Data-Classification-Framework
A Data Classification Framework is a governance model that organizes enterprise data based on sensitivity, risk impact, and regulatory requirements
# Enterprise Data Classification Framework

This repository provides a structured framework for enterprises to classify and protect sensitive data.

Organizations can implement data classification models such as:

- Public
- Internal
- Confidential
- Restricted

This framework helps CISOs improve data governance and compliance.
